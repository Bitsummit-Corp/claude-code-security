# /ccsec - BITSUMMIT Hardening

Apply the BITSUMMIT Hardening profile to your Claude Code configuration.

Usage:
- `/ccsec apply baseline` - solo-dev profile
- `/ccsec apply strict` - team/shared-infra profile
- `/ccsec apply regulated` - healthcare/legal/public-sector profile
- `/ccsec doctor` - diagnose current installation

This command delegates to the `ccsec` CLI from `@bitsummit/ccsec-cli`.
